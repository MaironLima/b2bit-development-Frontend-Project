## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/MaironLima/Mini-Twitter-Project.git
cd Mini-Twitter-Project
npm install
# or
yarn install
```

### Running the Project

```bash
npm start
# or
yarn start
```

### Building for Production

```bash
npm run build
# or
yarn build
```
