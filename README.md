## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (recommend v16+)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/MaironLima/b2bit-development-Frontend-Project.git
cd b2bit-development-Frontend-Project
npm install
# or
yarn install
```

### Running the Project

```bash
npm start
# or
yarn start
```

### Building for Production

```bash
npm run build
# or
yarn build
```
